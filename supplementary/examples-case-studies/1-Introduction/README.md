Analysis for the illustrative example presented in Section 1 (Introduction). We provide two different versions: 

1. [Analysis with Repeated-Measures ANOVA](https://www.journalovi.org/2024-tsandilas-ranktransforms/supplementary/examples-case-studies/1-Introduction/illustrative-example-aov.html) using R's ```aov``` function.
